# Music Review Website

## Overview
This project is a dynamic website where users can review their favorite songs, albums, and artists. Built using **HTML**, **CSS**, and **Python** with Sphinx, it allows users to leave comments, ratings, and share their favorite music genres.

## Features
- User-friendly interface for submitting reviews
- Star-rating system
- Categories for genres, albums, and artists
- Backend integration using Python for handling submissions

## Technologies Used
- Python (Flask for backend)
- HTML/CSS
- JavaScript for interactivity
