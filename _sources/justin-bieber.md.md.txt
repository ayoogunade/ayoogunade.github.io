# Justin Bieber Fanpage

## Overview
This fanpage is dedicated to Justin Bieber, showcasing his biography, discography, and career highlights. The website also includes a gallery, trivia, and an interactive timeline of his musical journey.

## Features
- Biography and career timeline
- Interactive media gallery
- Fun facts and trivia quiz
- Discography with album covers and links

## Technologies Used
- Python (Sphinx)
- HTML/CSS for layout and styling
- JavaScript for interactive elements
